# Weather App 

   Bu repo [Kodluyoruz](https://www.kodluyoruz.org/) ve [Patika.dev](https://app.patika.dev/) tarafından verilen **Weather App** React uygulamasını içermektedir. Bu uygulama sayesinde input'a yazdığımız şehir isminin 3 günlük hava durumu tahminlerini görebiliyoruz.
